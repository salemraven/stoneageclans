# STONE AGE GAME MUSIC DOCTRINE

## Mission: Create Fully Primitive-Sounding Music for a Stone Age Video Game

---

# 1. CORE MISSION

We are creating music that:

* Sounds deeply prehistoric
* Uses only primitive instruments
* Avoids ALL modern musical influence
* Feels communal, ritualistic, breath-driven
* Could plausibly exist in the Paleolithic
* Supports a Stone Age survival/ritual/raiding atmosphere

This is NOT:

* Tribal EDM
* Cinematic Hollywood tribal
* Orchestral hybrid
* World-music fusion
* Clean modern production

If it sounds modern, it failed.

---

# 2. HARD INSTRUMENT RULES

## Allowed Instruments Only

* Animal-hide drums (frame drums, hand drums)
* Hollow logs
* Wooden sticks / clapsticks
* Stone percussion (rocks struck together)
* Seed or pebble shakers
* Bone flutes (breathy, imperfect tone)
* Conch or animal horn blasts
* Primitive wooden drone (breath-driven, raw)

## Absolutely Forbidden

* Synths
* Electronic percussion
* Modern drum kits
* Snare-driven military march
* 4-on-the-floor rhythm
* Cinematic trailer percussion
* Orchestral taikos
* Clean reverb-heavy production
* Dance groove of any kind

Beats may be precursors to modern rhythm — but must NEVER resemble a modern beat.

Never dance beats.

---

# 3. RHYTHM DOCTRINE

## 3.1 Rhythm Must Be Human

* Uneven
* Breath-paced
* Step-driven
* Slightly imperfect
* Organic timing drift

No metronomic perfection.

Drums should feel:

* Hand-struck
* Body-powered
* Earth-connected

---

## 3.2 Vocals Control the Beat (CRITICAL RULE)

The voice drives rhythm.

Drums follow the chant.

Not the other way around.

Dragged syllables dictate spacing between drum hits.

---

## 3.3 No Modern Groove

If it resembles:

* Euro dance
* EDM
* Clean marching band
* Club rhythm
* Clean loop pattern

It must be corrected at the PROMPT level.

If modern beats appear:
The prompt is wrong — not the lyrics.

---

# 4. LYRIC STRUCTURE RULES (NON-NEGOTIABLE)

## 4.1 4-Beat Line Structure

Preferred structure:

Each line contains FOUR beat clusters.

Example structure:

```id="u9x4kp"
Word… Word… Word… Word…
Word… Word… Word… Word…
Call… Response… Call… Response…
Repeat… Repeat… Repeat… Repeat…
```

Each line should feel like:

STEP STEP STEP STEP

This is a hard rule.

---

## 4.2 Repetition Is Mandatory — But Controlled

Repetition builds trance.

However:

Do NOT spam identical syllables endlessly.
Do NOT reuse the same rhythmic phrase every song.

Bad:

```id="h5mz8t"
Duuuum… Duuuum… Duuuum… Duuuum…
```

Better:

```id="t4rq6n"
Duuum… Uum… Dum… Uum…
```

Best:

```id="q2lp7s"
Duuum… Uum… Dum… Hrrr…
```

Patterned repetition.
Slight variation.
Human imperfection.

---

## 4.3 Dragged Syllables Control Tempo

Use elongated vowels and consonants:

* Hrrr…
* Rrruuu…
* Duuum…
* Kaaa…
* Shhhh…
* Ooooh…
* Hhhhhh…

Purpose:

* Slow pacing
* Control drum spacing
* Create hypnosis
* Simulate breath

Never overuse one dragged sound repeatedly across songs.

Mix it up rhythmically:
Duuum… Uum… Dum… Hrrr…

---

## 4.4 Call and Response (Strongly Preferred)

Leader line.
Group echo.

Example:

```id="j8q4nr"
Fire rise… Fire rise…
We walk slow… We walk slow…
```

Or proto-form:

```id="m3v9kt"
Garra… Garra…
Dhaa… Dhaa…
```

This builds communal energy.

---

## 4.5 Language Requirements

Lyrics must feel:

* Elemental
* Physical
* Earth-based
* Ritualistic
* Pre-literate
* Direct

Allowed:

* Proto-syllables
* Simple primal English
* Nature imagery
* Ritual imagery
* Breath sounds

Avoid:

* Complex poetic metaphors
* Abstract philosophy
* Modern slang
* Modern sentence complexity
* Overly repetitive nonsense syllables with no structure

There must be a sense of intentional language — not random noise.

---

# 5. REGION-SPECIFIC AUTHENTICITY

When doing regional themes:

* Reinforce primitive instruments specific to environment
* Avoid stereotypical modern “tribal” production
* Respect plausibility
* Avoid modern cultural fusion

Example considerations:

Jungle:

* Log drums
* Shell horns
* Breath echoes

Desert:

* Clapsticks
* Deep drone
* Sparse percussion

Tundra:

* Wind-like vocals
* Low drums
* Minimal percussion

Neanderthal:

* Throaty consonants
* Chest resonance
* Stone percussion

---

# 6. HYPNOSIS FORMULA

Hypnotic effect comes from:

1. 4-beat repetition
2. Dragged syllables
3. Call and response
4. Layered voices
5. Density increase (NOT tempo increase)

Never increase speed to build intensity.

Increase:

* Vocal layers
* Drum clusters
* Drone thickness
* Group chanting strength

---

# 7. SUNO BEST PRACTICES

## 7.1 Keep Prompts Short

Suno has character limits.

Use minimal descriptive triggers.

---

## 7.2 Prompt Formula

Structure:

Theme.
Primitive instrument list ONLY.
No modern elements.
Organic rhythm description.
Vocal structure description.
Atmosphere.

Example:

```id="f7m3pl"
Stone Age ritual chant.

Animal-hide drums, wooden sticks, stone percussion, bone flute, seed shakers. No modern instruments.

Organic uneven rhythm. No dance groove. Vocals drive percussion.

4-beat repetitive chant with dragged syllables and call and response.

Raw fire-lit communal ceremony.
```

---

## 7.3 If Modern Beats Appear

DO NOT rewrite lyrics first.

Fix the prompt.

Remove words like:

* Driving
* Powerful
* Percussive groove
* Epic
* Tribal energy
* Rhythmic pulse

These trigger modern production.

---

# 8. QUALITY CONTROL CHECKLIST

Before finalizing, confirm:

* Only primitive instruments listed
* No modern rhythmic language in prompt
* 4-beat line structure present
* Repetition used intentionally
* Dragged syllables control tempo
* Vocals drive drums
* No dance feel
* No cinematic feel
* No modern groove
* Feels plausibly Paleolithic

If all true → It is “by the book.”

# 9. FINAL STANDARD

The music should feel like:

Firelight.
Breath in cold air.
Wood striking wood.
Stone striking stone.
Voices rising before drums.
Rhythm born from walking.
Hypnosis from repetition.
Older than written language.

If it feels like a festival track, it failed.

If it feels like humans in deep time, it succeeded.

RTS background music must:

## 2️⃣ Instrument Palette (Authentic but Game-Ready)

**Percussion**

* Frame drum (animal hide tone)
* Hollow log drum
* Stone hits
* Bone clicks
* Seed / pebble shaker
* Low tribal stomp rhythm

**Wind**

* Bone flute
* Reed pipe
* Low horn calls
* Airy breath textures

**Vocal Layers**

* Drone layer (low sustained vowel)
* Mid rhythmic chant layer
* High airy overtone layer
* Call-and-response motif layer

We use harmony through:

* Drone + moving melody
* Parallel vocal movement
* Rhythmic layering

## 3️⃣ Structural Strategy by Game State

| Game State              | Best Structure |
| ----------------------- | -------------- |
| Early build phase       | AAA            |
| Steady gameplay         | AB             |

| Cinematic ritual        | AABA           |
| Exploration / expansion | ABC            |

---

## 4️⃣ Loop Engineering

To make it game-ready:

* End phrases resolve softly, not sharply
* Avoid hard cadence endings
* Use repeating vowel tones at end
* Keep tempo stable
* Avoid massive dynamic spikes

---

## 5️⃣ Emotional Tone Control

**Calm expansion**

* More flute
* Lighter percussion
* Air vocals

**War preparation**

* Stronger drum pulse
* Throat-style low drones
* Shorter vocal phrases

**Combat intensity**

* Denser rhythm
* Call-and-response
* Repeating chant hooks

---

## 6️⃣ Identity Consistency

Everything should feel like it comes from the same world:

* Same vowel palette
* Same rhythmic language
* Same drum texture
* Same harmonic base

That’s how you create a recognizable Stone Age RTS identity.
