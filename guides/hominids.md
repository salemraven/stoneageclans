### Hominid Classes & Trait Hybridization Mechanic

Let's brainstorm this rich hybridization mechanic to make hominids feel like distinct "races" that evolve your clan over generations. We'll consolidate from our chat (stats/traits like strength/agility for Neanderthal/Floresiensis) and npcguide.md (modular traits, stat modifiers, reproduction). The goal is a simple, rich system where babies mix parent traits 50/50, creating unique hybrids that unlock new playstyles (e.g., a sapiens-neanderthal mix = smart tanks). Traits are modular (add/remove in dev menu), affect stats/FSM priorities, and show in Clan Menu (C key) with icons.

#### All Hominid Classes (5 Total – Base Ones from Chat)
Each has 3–4 unique traits (stat buffs, behavior tweaks). Base stats 50/100 – traits add +X % (editable). Visuals: Slight sprite tweaks (e.g., Neanderthal stocky, Floresiensis small).

1. **Homo sapiens** (Adaptable social leaders – good all-rounders for early clans)
   - Traits:
     - "Learning" (+20 % Intelligence – faster tier progression, craft speed)
     - "Social Strategist" (+15 % Social – better herding/mating success, +10 % clan morale)
     - "Adaptive" (+10 % Agility – +herd chance, dodge in combat)
     - Role: Balanced builders/reproducers.

2. **Neanderthal** (Tough warriors – tanky for defense/raids)
   - Traits:
     - "Robust" (+20 % Strength, +15 % Endurance – melee damage, wound resistance)
     - "Pain Resistant" (+25 % Pain Tolerance – shorter debuff duration)
     - "Hunter's Grit" (+10 % Aggression – higher attack vs flee chance)
     - Role: Frontline fighters, hunters.

3. **Heidelbergensis** (Sturdy workers – resource machines)
   - Traits:
     - "Enduring Laborer" (+20 % Stamina, +15 % Carry Capacity – longer trips, more loot)
     - "Builder's Might" (+10 % Intelligence – faster build/craft)
     - "Resilient" (+10 % Endurance – less hunger depletion)
     - Role: Gatherers, builders.

4. **Denisovan** (Keen hunters – aggressive scouts)
   - Traits:
     - "Sharp Senses" (+20 % Perception – longer detection range for resources/enemies)
     - "Aggressive Predator" (+15 % Aggression – higher hunt/attack priority)
     - "Nomad Endurance" (+10 % Pain Tolerance – resist wounds/debuffs)
     - Role: Hunters, raiders.

5. **Floresiensis** (Agile swarmers – fast breeders)
   - Traits:
     - "Swarm Instinct" (+20 % Social – better flocking/herding, +10 % reproduction chance)
     - "Nimble" (+15 % Agility – higher speed/dodge)
     - "Compact" (+10 % Fertility – faster baby growth)
     - Role: Breeders, swarm attackers.

#### Rich Hybridization Mechanic
- **Mixing Rule**: Babies inherit 50/50 traits from parents (e.g., sapiens + neanderthal = "Learning" + "Robust" traits, averaged stats like +10 % Intelligence +10 % Strength).
- **Trait Inheritance**: Random 50% chance per trait from each parent (up to 6 traits max per NPC).
- **Stat Blending**: Average parent stats (e.g., sapiens 50 Intelligence + neanderthal 40 = 45 for baby).
- **Quality Tier Overlay**: Age-based tiers multiply hybrid stats (Flawed -20%, Legendary +60%).
- **Display**: Clan Menu shows "Hybrid: Sapiens 50% / Neanderthal 50%" + blended traits/icons.
- **Balancing**: Dev menu sliders for inheritance chance, max traits. Hybrids unlock "unique combos" (e.g., sapiens-floresiensis = smart swarmers).

This makes clans evolve: Start sapiens → herd neanderthal woman → tough smart hybrids → dominate.

**Suggestions**: Add "trait mutation" (1% chance for new random trait on birth) for surprise. Hybrid art: Blend sprites (e.g., sapiens-neanderthal = taller with heavy brow).
